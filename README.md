# Tick Shield Advisor

A comparison tool to help users evaluate and rank tick prevention interventions based on effectiveness, environmental impact, and cost.

---

## 🚀 Getting started

### Install dependencies

```sh
npm install
```

### Run in development mode

```sh
npm run dev
```

Then open the URL shown in the terminal (usually http://localhost:5173).

### Build for production

```sh
npm run build
```

### Preview the production build

```sh
npm run preview
```

---

## 🧱 Project structure

- `src/` – application source code
- `public/` – static assets (favicon, images, etc.)
- `dist/` – production build output (generated)
- `src/data/` – intervention data + images
- `src/components/` – UI components and pages

---

## 🧪 Tests

This project uses Vitest for unit testing.

Run tests with:

```sh
npm test
```

---

## 🔧 Notes

- The data model for interventions lives in `src/data/interventions.ts`.
- The favicon is stored in `public/favicon.ico` and referenced from `index.html`.
